六种css水平垂直居中方法

发表于 2018-07-15 | 分类于 前端学习
字数统计: 489 | 阅读时长 ≈ 2

俗话说，好记性不如烂笔头。七月中旬了，马上要迎来秋招，打算在博客里做一下关于前端的一些笔试、面试的笔记，以及梳理一下前端的一些知识，就从最简单但是也问的最多的垂直居中写起吧，加油！坚持就是胜利。

利用绝对定位与transform

<div class="parent">

    <div class="children"></div>

</div>

对应的CSS为：

.parent{
    height:800px;
    width:800px;
    border:1px solid #000;
    position:relative;
}
.children{
    width:500px;
    height:300px;
    position:absolute;
    top: 50%;
    left: 50%;
    transform:translate(-50%,-50%);
    border:1px solid #000;
}

用margin实现

当子元素的宽高固定，父元素内含有除居中元素外其它元素（空标签也行）或者父元素的高度不为0时,将父元素定位，子元素绝对定位，利用margin负值为子元素宽高的一半来实现。

<body> 
    <div></div>
</body>

以下是控制div居中的css:

html,body{
    width:100%;
    height:100%;
}
div{
    position:absolute;
    width:500px;
    height:300px;
    left:50%;
    top:50%;
    margin-left:-250px;
    margin-top:-150px;
    border:1px solid #000;

}

利用定位与margin：auto

<div class="parent">
　　<div class="child">哈哈</div>
</div>

css代码：

.parent{
        width: 600px;
        height: 400px;
        background: red;
        position: relative;
        }
.children{
       width: 200px;
       height: 200px;
       position: absolute;
       top: 0;
       left: 0;
       bottom: 0;
       right: 0;
       margin: auto;
       background: black;
         }

利用flexbox

　　.parent{

　　　　justify-content:center;
　　　　align-items:center;
　　　　display: -webkit-flex;

　　}

利用table

table的td默认内容垂直居中。只要再设置内容水平居中即可

display:table-cell

CSS中有一个用于竖直居中的属性vertical-align，但只有 当父元素为td或者th时，这个vertical-align属性才会生效，对于其他块级元素，例如 div、p等，默认情况下是不支持vertical-align属性的，可以设置块级元素的display类型为table-cell，激活vertical-align属性，但display:table-cell存在兼容性问题，所以这种方法没办法跨浏览器兼容。

　　<div class="parent">
　　　　<div class="child">哈哈</div>
　　</div>

css代码：

　　.parent{
　　　　width: 400px;
　　　　height: 100px;
　　　　background: black;
　　　　display: table-cell;
　　　　vertical-align: middle;
　　　　text-align: center;
　　}

　　.child{
　　　　background: red;
　　　　display: inline-block;

　　}
