Start

发表于 2018-05-02 | 分类于 随笔

呼~~个人博客终于大致搭建起来了，一开始萌生搭建个人博客的念头还是和另一个学前端的小伙伴聊天时听她说想搭一个来记录自己的学习过程，看着和听着觉得挺简单，没想到自己动手开始做的时候才发现那么多坑，就说博客主题吧，改呀改终于差不多能看了，但是还是有一些想改进的地方，毕竟是用的大神做好的Next主题,和自己一开始想完全自己做博客页面还是不一样的，得看着Next文档和一些主题优化文章边学边改，有时候很烦的是还改错了，又得回头找再改，总之以后再慢慢改吧，有种像搭房子的感觉，房子虽然建起来了，装修还要好好弄呢。

第一篇博客写着写着突然不知道怎么写了，总之还要谢谢小小同学了，这里戳上她的博客（大家可以去帮忙去增加访客量哟~），之前在阿里云注册了自己的域名之后，就卡在怎么把自己的域名和GitHub Pages关联起来，踩了好多次坑，被弄的没点心情了，要不是她督促了一下，我这个拖延症重度患者不知道什么时候才能把这个小窝搭好。接触新事物才能学习新东西，虽然踩坑真的很让人抓狂，尤其是半天解决不了的时候，百度各种方法尝试，现在回想起来发现其实蛮简单的，都是一些配置的问题，打算有时间了写一篇关于怎么用Hexo + GitHub搭建个人博客的文章吧，把这几天踩坑是怎么解决的也总结一下，希望后来者别如我这般悲催，祝大家都能一次性搭好。

后面还要再看看Hexo文档和熟悉Markdown的写作，总之熟能生巧，有一大堆的关于自己学习前端的总结要写呢，好记心不如烂笔头，希望能养成更新博客的好习惯，共勉。
