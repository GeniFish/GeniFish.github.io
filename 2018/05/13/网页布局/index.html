网页布局

发表于 2018-05-13 | 分类于 前端学习
字数统计: 1,476 | 阅读时长 ≈ 6

前言

网页实质是块与块之间的位置，块挨着块，块嵌套块，块叠着块。

三种关系：相邻，嵌套，重叠

下面介绍网页布局的常用几种方式

1.一列布局

这个so so esay，但是还是做一下笔记吧。一般都是固定的宽高，设置margin : 0 auto来水平居中，用于界面显著标题的展示等；

.main{
            width: 200px;
            height: 100px;
            background-color: grey;
            margin: 0 auto;
        }
<div class="main"></div>

2.两列布局

说起两列布局，最常见的就是使用float来实现。float浮动布局的缺点是浮动后会造成文本环绕等效果，以及需要及时清除浮动。

设置左左浮动（需要确定父级元素的宽度）

如果父级元素没有设置高度，则需要设置overflow:hidden来清除浮动产生的影响，因为浮动的子元素不能撑开父盒子，父盒子设置overflow:hidden才能根据子元素高度自适应。

对于自己相邻元素清除浮动产生的影响用：clear：both。

<div class="main">
    <div class="left">left</div>
    <div class="right">right</div>
</div>

CSS代码

 .main{
    width: 400px;
    background: red;
    overflow: hidden;
}

.left{
    background: yellow;
    float: left;
}

.right{
    background: green;
    float: left;
}

3.三列布局

两侧定宽中间自适应

用绝对定位实现

但是效果图是酱紫的：

虽然能实现两侧的div固定宽度不变中间的div随着浏览器宽度变化而变化，但是右边的蓝色DIV为什么会掉下来嘞？
我们改变一下HTML中这三个div的排列顺序看看

<div class="left">left</div>
<div class="right">right</div>
<div class="middle">middle</div><!-- 自适应的DIV一定要放在浮动DIV后面 -->

是不是觉得很神奇，下面是一大堆解释：

xhtml中包含两种元素：块级元素（如div、p、h、列表、table、form、body）和内联元素（如span、a、img、input）

块级元素：就是一个方块，像段落一样，默认占据一行出现；

内联元素：又叫行内元素，顾名思义，只能放在行内，就像一个单词，不会造成前后换行，起辅助作用。

一句话归纳就是：浮动就是个带有方位的display:inline-block属性，而inline-block是将对象呈递为内联对象，但是对象的内容作为块对象呈递。旁边的内联对象会被呈递在同一行内。

这样就能解释为什么蓝色块会掉下来了，红块采用的是float，黄块紧随其后，被显示在同一行，而黄色默认的float:none，即不带有float属性，作为块级元素，蓝块就会换行。

Flexbox实现

也叫做圣杯布局格式：页面从上到下为头部、中部、脚部；头部、脚部定高，不可伸缩；中部高度自适应。
中部三列式布局：左右两列定宽，不可伸缩；中间内容区自适应。

CSS代码：

<style type="text/css">
    *{
        box-sizing:content-box;/* 伸缩项目自动box-sizing:border-box，所以需调整为content-box */
        margin:0;
        padding:0;
    }

    body{
        display:flex;
        flex-direction:column;/* 头、中部、脚纵向显示 */
    }

    header,footer{
        flex:0 0 50px;/* 头、脚尺寸固定，不放大、不缩小 */
        background:#3f3f3f;
    }

    .main{
        display:flex;

        /* 
        flex:1 == 1 1 auto：剩余空间放大比例(flex-grow)  空间不足缩小比例(flex-shrink)    分配多余空间之前项目占据的主轴空间(flex-basis)
        flex:1指的是：中部区域自由伸缩
        auto指的是项目本来大小，因未给main设置高度,main高度由子元素最高者决定，若子元素高度为0，则main高度为0
        块级元素未主动设置高度或未被子元素撑起高度，浏览器默认为块级元素分配高度为0。
        */
        flex:1;
    }

   .content{
        background:red;
        height:1000px;

        /* 
        横向中间内容区自适应，即使未指定宽度，但会分配宽度 
        块级元素未主动设置宽度或未被子元素撑起宽度，浏览器默认为块级元素分配宽度为可使用的全部宽度，比如全屏宽。
        */
        flex:1;
   }
   .left,.right{
        height:800px;
        background:blue;
        flex:0 0 100px;/* 左右两列固定宽 */
   }

   .left{
        order:-1;/* 让left居于左侧 */
   }

</style>

HTML代码：

<body>
    <header></header>
    <div class="main">
        <div class="content"></div>
        <div class="left"></div>
        <div class="right"></div>
    </div>
    <footer></footer>
</body>

4.混合布局

混合布局其实就是前面三种一起用而已啦
还是举个栗子吧

CSS代码：

.top{
    height: 100px;
    background: teal;
}
.footer{
    height: 100px;
    background: wheat;
}
.main{
    width: 100%;
    background: red;
    overflow: hidden;
}
.left{
    background: yellow;
    float: left;
    width: 50%;
}
.right{
    background: green;
    float: right;
    width: 50%;
}

HTML代码：

<div class="top"></div>
<div class="main">
    <div class="left">left</div>
    <div class="right">right</div>
</div>
<div class="footer"></div>
